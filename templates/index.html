    ℹ️ How It Works

    🔗 URL Analysis

    The model analyzes 8 key features of URLs including length, domain characteristics, and special patterns.

    🤖 Machine Learning

    Uses a Random Forest classifier trained on legitimate and phishing URLs for accurate detection.

    ⚡ Real-Time

    Get instant predictions with confidence scores to help you make informed browsing decisions.

    🔒 Privacy

    Analysis happens locally on your server. URLs are not stored or logged permanently.

    📊 Analyzed Features

    • URL Length: Phishing URLs tend to be longer
    • Dots Count: Multiple dots can indicate suspicious domains
    • Hyphens in Domain: Legitimate domains rarely use hyphens
    • @ Symbol: Presence often indicates spoofing attempts
    • HTTPS Protocol: Legitimate sites use secure HTTPS
    • Domain Length: Unusual domain lengths may be suspicious
    • Numeric Ratio: High numeric content can be suspicious
    • Special Characters: Excessive special characters are suspicious
